Prembulo

Cuando se utiliza la regularizacin:

  • Problema de prediccin con una alto nmero de predictores (overfitting)
  • Mucho ruido en la base de datos (muchos predictores no estn relacionados con la respuesta)
  • Alta correlacin entre los predictores (multicolinealidad)

Porqu Regularizar?

  • La estimacin de los coeficientes puede ser muy inestable (altas varianzas)
  • Permite una seleccin continua de variables
  • Subsana los problemas que conlleva la multicolinealidad

En qu consiste la regularizacin?

Regresin lineal:

Para \(i=1,2,\dots,N\),

\[ y_i = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \dots + \beta_K x_{iK} +\varepsilon_i\]

donde \(\varepsilon_i \sim N(0,\sigma^2)\).

los coeficientes \(\beta_k\) se estiman sin restriccin.

Regresin con regulariazin:

los coeficientes \(\beta_k\) estn restringido.

  1. Ridge regression: \[\sum_{k=1}^K \beta_k^2 \leq t\]
  2. Lasso regression: \[\sum_{k=1}^K \left| \beta_k\right| \leq t\]
  3. Elastic-net: mixtura de Ridge y Lasso \[\sum_{k=1}^K \left[ \alpha\left| \beta_k\right| + (1-\alpha) \beta_k^2 \right] \leq t\]

Interpretacin de la regularizacin:

  • Shrinkage:
    • Ridge: asemeja los coeficientes de los predictores mutuamente correlacionados
    • Lasso: selecciona uno de estos predictores y descarta los dems.
  • PCA: Penaliza las direcciones (combinaciones) de los predictores poco informativas.
  • Bayesian:
    • Ridge ~ los coeficientes tienen una distribucin (a priori) gaussiana.
    • Lasso ~ los coeficientes tienen una distribucin (a priori) de Laplace.

Overfitting

Para ilustar este problema, seguimos con el ejemplo de los trenes.

n=100
DT=data.table(distancia=rnorm(n,0,3)) #distancias estandarizadas
DT[,media:= 2 + exp(distancia)/(1+exp(distancia))]
DT[,retrasos:= media + rnorm(n,0,.5)]
p<-ggplot(DT,aes(y=media,x=distancia)) + geom_line(size=1,color="blue4") + 
  ylim(range(DT$retraso))+xlab("distancia estandarizada") 
p

Para investigar la capacidad predictiva del modelo, partimos la base de datos en un conjunto de adiestrameinto y un conjunto de test:

DT[,muestra:=sample(c("train","test"),n,prob=c(2/3,1/3),replace=TRUE)]
p<- p + geom_point(aes(y=retrasos,color=muestra),DT)
